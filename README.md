# 🥤 Pepsi Landing Page

A sleek and modern **Pepsi-inspired landing page** built with HTML, CSS, and JavaScript.

Landing page ini menampilkan desain yang clean, interaktif, dan dilengkapi animasi halus untuk meningkatkan user experience.

---

## ✨ Highlights

- 🎨 Modern UI & clean layout  
- ⚡ Smooth animations & transitions  
- 📱 Fully responsive design  
- 🖱️ Interactive elements  
- 🚀 Lightweight & fast  

---

## 🧰 Tech Stack

- HTML5  
- CSS3  
- JavaScript

## ⚙️ Getting Started

Jalankan project ini di lokal dengan langkah berikut:

1. **clone repo**
```bash
git clone https://github.com/gangsarrz/LandingPage-Pepsi.git
```
2. **masuk ke folder**
```bash
cd LandingPage-Pepsi
```
3. **jalankan**

   buka ```index.html ```di browser
